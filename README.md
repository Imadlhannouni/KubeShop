# Kubernetes Zero to Kuberstronaut: KubeShop Labs

Welcome to the **KubeShop** learning path! This project contains the source code and lab exercises to take you from a Kubernetes beginner to an expert "Kuberstronaut".

## Project Structure

- **src/**: The source code for the microservices application you will be deploying.
  - `catalog/` (Go): Product catalog API
  - `frontend/` (Node.js): Web interface
  - `order/` (Python): Order processing service
  - `payment/` (Go): Mock payment gateway
- **labs/**: The hands-on exercises. Each folder contains instructions and starting files.
- **scripts/**: Helper scripts (if any).

## How to Start

Navigate to the `labs/` directory and start with **Lab 01**.

```bash
cd labs/01-docker
cat instructions.md
```

Follow the instructions in each lab sequentially. Good luck!
